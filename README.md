# Role Faction Changer

A simple mod to simplify the way you mention roles in a different faction, such as a shourd alligned admirer

# How to use

There are currently 2 ways to mention a role, in all cases you will want to use the role itself (by typing #(rolename) and pressing TAB or ENTER to confirm). The currently avaliable methods are:

* **\[\[ROLE,FACTION\]\]**, example: \[\[#Admirer,#Shroud\]\].
* **ROLE(FACTION)**, example: #Admirer(#Shroud).
